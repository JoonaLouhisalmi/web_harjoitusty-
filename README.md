# Sääsovellus

Tämä on yksinkertainen sääsovellus, joka hakee reaaliaikaista säätietoa OpenWeather API:n avulla.

## Ominaisuudet

- Hae säätiedot kaupungin nimellä
- Näytä lämpötila, säätila ja tuulen nopeus
- Sääikonit OpenWeatherilta
- Tallentaa suosikkikaupungit selaimen muistiin (localStorage)
- Muistaa viimeksi haetun kaupungin
- Responsiivinen käyttöliittymä eri näyttökokoihin

## Teknologiat

- HTML
- CSS
- JavaScript
- OpenWeather API
- LocalStorage

## Käyttö

1. Avaa sivu selaimessa
2. Kirjoita kaupungin nimi hakukenttään
3. Paina "Hae"
4. Säätiedot näytetään sivulla
